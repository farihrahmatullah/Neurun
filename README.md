# Neurun

Neurun adalah aplikasi belajar Hiragana dan Katakana yang dirancang untuk membantu pemula menguasai huruf Jepang melalui latihan cepat, interaktif, dan berulang. Tidak ada teori panjang pengguna langsung belajar lewat pengenalan bentuk huruf, penyusunan kata, dan respon instan.

Fokus utama project ini adalah implementasi logika gameplay, pengelolaan state permainan, serta struktur project Unity yang rapi dan terkontrol.

- Penyusunan kata yang benar akan menambah skor
- Kesalahan memilih huruf atau terkena musuh dapat mengurangi skor atau mengakhiri permainan
